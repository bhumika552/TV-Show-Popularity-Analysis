📺 TV Show Popularity Analysis using Data Mining
🚀 Overview

This project focuses on analyzing TV show popularity using data mining and sentiment analysis techniques. It collects user feedback (comments) along with demographic details and generates insights through interactive visualizations.

The system classifies user sentiments (Positive/Negative) and calculates a popularity score for each TV show.

🎯 Features
📊 Sentiment Analysis on user comments
👥 Demographic analysis (Age, Gender, Location)
📈 Interactive dashboard with multiple graphs
🏆 TV show popularity ranking system
🖥️ GUI-based system (Admin & Visitor modules)
📂 Automatic dataset creation and management
🧠 Methodology
Collect user data (Name, Age, Gender, Location, Show, Comment)
Apply sentiment analysis using NLP
Classify comments into Positive/Negative
Compute popularity score:
Popularity Score = Positive Comments − Negative Comments
Visualize results using graphs
